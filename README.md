# guthoms_helpers
This repository contains the python helper tools i use in different projects. 

## Prerequisites
* Python 3.6 (for Typing etc.)
* SimpleJson
* Numpy
* OpenCV
* Progressbar2
* websockets
* PyTorch

## Installing (Source)
Clone the needed repositories to your virtual environment and install requirements:

git clone https://github.com/guthom/guthoms_helpers
cd guthoms_helpers
pip install -r requirements.txt
